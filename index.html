HTML

  • HTML structures the website's content!
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Inside the body element, there are header, main, and footer elements.
  • Content is organized by semantic elements such as paragraph, heading, and img elements.

CSS

  • CSS styles the website!
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The border of an element can be modified with width, color, and border style to highlight an element.
  • Class selectors can be used to apply a style to a series of selected elements regardless of element type.
  • CSS styles can be accessed in 3 ways: inline styles, internal stylesheets, and external stylesheets. External is the most popular and helps achieve separation of concerns.

Git

  • Git is a version control system that helps keep track of saved versions of software, and lets you share code with other developers and the world!
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add: stages changes to prepare for the next commit
  • git commit -m "(type a declarative message)": commits changes and gives them a code that can be reverted to if changes cause any unwanted behavior
  • git checkout HEAD . : reverts to the way files appeared as of the previous commit
  • git pull origin main: pulls the latest main branch to local machine you're coding on

JavaScript

  • Javascript adds interactivity to the website!
  • Javascript has various data types, such as numbers, boolean values (true or false), and strings of characters.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A loop lets you interrupt the control flow and access code using logical operators such as comparing the value of a number.
  • A function is a block of code that can be used as many times as you need it. It helps keep code modular and efficient.
  • console.log() is a method that lets the programmer print specific information at a point in the program to a console to help debug the code.
